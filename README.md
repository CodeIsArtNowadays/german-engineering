# 01. Автономный ИИ-Агент на чистом Python (Native ReAct)

В этом модуле я реализовал классический паттерн **ReAct (Reasoning + Acting)** без использования сторонних фреймворков (LangChain/LangGraph). Цель — понять "подкапотку" агентских движков и управление контекстом.

### Что под капотом:
* **Динамический цикл управления (`while`)**: Модель сама решает, когда ей нужно вызывать инструменты, а когда остановиться и выдать финальный ответ.
* **Parallel Tool Calling**: Поддерживается параллельный вызов нескольких инструментов за одну итерацию (например, запрос погоды сразу для 3 городов).
* **Строгая валидация**: Аргументы, которые генерирует LLM, автоматически проверяются на соответствие типов через схемы `Pydantic`.
* **Самоисправление ошибок**: Если инструмент падает или модель ошибается в JSON, код отлавливает исключение и отправляет лог ошибки обратно в контекст LLM, чтобы она исправила параметры на следующем шаге.

### Как запустить:
1. Создайте `.env` и добавьте `OPENAI_API_KEY`.
2. Запустите `python agent.py`.
